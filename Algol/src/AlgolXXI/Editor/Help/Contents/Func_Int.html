Parte inteira de um número

Sintax

inteiro int (real numero)

Descrição

retorna a parte inteira de um número

Exemplo

inicio
    real numero
    inteiro valor
    numero <- 10.9
    valor <- int ( numero )
    escrever "parte inteira(" , numero , ") = " , valor
fim

Resultado

parte inteira(10.9) = 10
