Constantes

Sintax

constante [tipo] [nome] <- [valor]
constante [tipo] [nome] <- [expressão]
constante [tipo] [nome] <- [valor] ,[nome] <- [expressão]

Definição

Define um símbolo cujo valor permanece inalterável durante os seu ciclo de vida.
Segue as mesmas regras que a definição de variáveis excepto que não é possível omitir o valor de inicialização

Exemplos

constante inteiro meses <- 12
constante real pi <- 3.14
