Definição de módulos (funções)

Sintax

tipo_de_retorno nome_do_modulo (parâmetros)

Definição

Para lidar com programas extensos é necessário dividi-los em módulos que possam ser analisados de forma independente. Uma função implementa uma parte do algoritmo, devendo ter poucas instruções, isto é, uma tarefa complexa é subdividida em diversas tarefas mais simples. Os módulos resultantes são facilmente compreendidos, testados, corrigidos e alterados. Uma função deve possuir uma única via de entrada e uma única via de saída. (Não é obrigatório existir Entrada ou Saída numa função.) Qualquer programa realizado no ALGOLXXI deve de conter sempre a função Inicio Programa, sendo esta a função principal do programa (equivalente à função main(...) de outras linguagens de programação). É nesta função que poderemos chamar qualquer outra que se tenha realizado. Todas as variáveis que tenham sido criadas dentro de uma função, são apenas conhecidas dentro dela.

Vantagens:

• Simplicidade: Cada sub problema é mais simples de resolver. No caso de funções usadas repetidamente, só é necessário escrevê-las uma vez.
• Manutenção: Reduzidos custos e tempos de alteração de um programa. As funções podem ser substituídas ou alteradas independentemente.
• Divisão de tarefas: Permite a distribuição das tarefas de programação pelos diferentes programadores.
• Eficiência: Permite a utilização de funções existentes em bibliotecas de software.

Como funciona uma função

• O código de uma função só será executado quando esta é invocada pelo programa principal.
• Sempre que uma função é chamada, o programa que a chama é suspenso de forma a executar todas as instruções presentes no corpo da função, assim que termina a função o controlo de execução do programa volta ao local em que a função foi chamada.
• O programa que chama a função pode enviar os argumentos para a função. Estes serão recebidos e armazenados em variáveis da função e inicializados com os valores enviados (Parâmetros) .
• Depois de terminar a função pode devolver um valor para o programa que a chamou.

• Nome: Designação pela qual a função é conhecida, obedece ás regras apresentadas para a definição de variáveis, devendo ser único.
• Parâmetros: Informação fornecida ao módulo de forma a parametrizar o seu comportamento, e estender a sua aplicabilidade. (Não é obrigatório existirem parâmetros numa função).
• Retorno: Tipo de valor que vai ser retornado pela função, caso esta não retorne nada deverá ser definida como “VAZIO”.

Exemplos

Inteiro dobro (inteiro x)
Retorna x+x
Fim dobro
