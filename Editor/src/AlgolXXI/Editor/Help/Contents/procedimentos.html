Procedimentos

Sintax

Vazio Nome_do_procedimento()
     ...
     Operacoes do procedimento
     ...
Fim Nome_do_procedimento
 

Definição

Uma função que não devolve qualquer valor é designada por procedimento. Sempre que uma função não devolve qualquer valor, deve de ser declarada como VAZIO, este método não tem retorno.

 

Exemplos

Vazio EscreverOla()
     Escrever “Olá”
Fim EscreverOla


Inicio Programa
      EscreverOla()
Fim Programa
